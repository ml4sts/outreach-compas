# Tutorial Logistics

## Usage

import the package
```
import compaslab
```

load the activity
```
tut = compaslab.Tutorial('filename')
```

initialize
```
tut.start()
```

proceed
```
tut.next()
```


## Adding a new tutorial

Add a notebook file to `compaslab/activities` directory in the [repository](https://github.com/ml4sts/outreach-compas)
